## 🎯 项目概述

### 战略定位

销售AI助手四期标志着从**"工具属性"向"代理属性"**的战略跨越。本项目基于Agentic AI理念，构建一个具备**感知、决策、执行与进化**能力的自主系统。

### 核心目标

| 维度 | 目标 | 预期提升 |
|------|------|----------|
| **数据资产化** | 聊天记录 → 结构化情报 | 减少1-2小时/天数据录入 |
| **能力标准化** | 销冠思维 → 团队资产 | 消除团队能力方差 |
| **流程自动化** | 被动响应 → 主动培育 | 线索转化率提升200% |
| **知识动态化** | 静态文档 → 自进化系统 | 知识更新周期从月→天 |

### 对标产品

- **Gong.io** - 对话智能分析
- **AutoSale** - 7天智能培育
- **Lavender** - AI邮件教练
- **Salesforce Einstein** - 预测性工作流

#### 技术实现

```typescript
// BANT提取Agent伪代码
class BANTExtractionAgent {
  async extract(conversation: Conversation): Promise<BANTData> {
    // 1. 上下文窗口构建
    const context = this.buildContext(conversation);

    // 2. LLM提取
    const raw = await this.llm.extract(context);

    // 3. 置信度评分
    const scored = this.scoreConfidence(raw);

    // 4. 人工确认阈值 (<85%)
    if (scored.confidence < 0.85) {
      return this.flagForReview(scored);
    }

    return scored;
  }
}
```

#### 知识新鲜度算法

```python
def calculate_freshness_score(knowledge_item):
    """计算知识新鲜度 (0-100)"""
    score = 100

    # 时间衰减
    days_since_update = (now - knowledge_item.updated_at).days
    score -= days_since_update * 0.1

    # 使用频率加成
    score += knowledge_item.usage_count * 0.5

    # 负反馈惩罚
    score -= knowledge_item.negative_feedbacks * 5

    return max(0, min(100, score))
```
