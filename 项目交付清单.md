## 🏗️ 技术架构

### 前端技术栈

```
React 18.3 + TypeScript 5.3
├── 状态管理: Zustand + React Query
├── UI框架: Shadcn/ui + Radix UI
├── 样式: Tailwind CSS 3.4
├── 图表: Recharts + D3.js
├── 富文本: Tiptap
├── 实时通信: Socket.io Client
└── 构建工具: Vite 5.0
```

### 后端技术栈

```
NestJS 10.3 + TypeScript 5.3
├── ORM: Prisma 5.8
├── 队列: BullMQ 5.1
├── 缓存: Redis 7.2
├── AI编排: LangChain
├── LLM: GPT-4o / Claude 3.5 Sonnet
├── 向量数据库: Milvus / Pinecone
├── 认证: JWT + Passport
└── 文档: Swagger/OpenAPI 3.0
```

### 核心功能模块

已设计的AI智能体：

4. **剧本生成Agent** - 场景化剧本
5. **RAG系统** - 知识库检索增强
